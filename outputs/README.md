# outputs/

Generated files from pipeline runs.

| File | Description |
|---|---|
| `eda_report.txt` | Full EDA report — dataset statistics, skill distributions, title counts |
| `ranking_run_log.txt` | Debug log from the most recent ranking run (top-10 with all scores) |
| `ranking_run_log_no_semantic.txt` | Run log without semantic scoring (for comparison) |
| `full_feature_scores.csv` | Feature scores for all 100K candidates (7.1 MB) |
| `embeddings.pkl` | Pre-computed MiniLM embeddings (generated by precompute_embeddings.py) |

`embeddings.pkl` and `full_feature_scores.csv` are in `.gitignore`.
